Helps getting Google Antigravity working smoothly

## Projects

### ConfirmScout

A Windows 11 WPF (.NET 8) application that monitors windows for confirmation buttons using OCR and provides keyboard shortcuts for automated interaction.

**Key Features:**
- Window selection and capture (2-5 FPS)
- OCR detection of "Confirm", "Continue", "Approve" buttons
- 2-frame stability requirement
- Always-on-top overlay highlighting
- Hotkey-based clicking (Ctrl+Alt+Enter) with safety checks
- Scroll search functionality (Ctrl+Alt+S)

See [ConfirmScout/README.md](ConfirmScout/README.md) for detailed documentation.
